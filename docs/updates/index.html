Updates & Support

How Hindsight keeps itself up to date, how to check manually, and where to report problems.

Hindsight updates itself. New versions download and install in the background, so you generally do not need to do anything to stay current.

Automatic updates

Hindsight checks for updates on its own, once a day by default. When a new version is available it can download and install it for you, then apply it the next time it is safe to do so.

You can tune this in the Colophon under The Apparatus (open the Index with ⌘K → Colophon, then go to the Apparatus section):

  • Automatically check for updates — on by default. Hindsight checks daily.
  • Automatically download updates — when on, a found update is fetched in the background so it is ready to install. This is available only while automatic checks are on.

Turn either off if you would rather decide when to update.

Checking manually

You can look for a new version at any time:

  • Hindsight → Check for Updates… in the menu bar, or
  • Colophon → The Apparatus → Check for Updates…

If a newer version exists, Hindsight shows you what changed and offers to install it. If you are already current, it tells you so.

Updates and your meetings

Hindsight never interrupts a recording to update. Update checks are paused while a meeting is recording, and an already-downloaded update will not relaunch the app mid-recording. Stop the recording and the update can proceed — a manual check will remind you if a recording is in progress.

Support and feedback

Found a bug, or have an idea? Everything lives in one place:

  • Help → Hindsight Help or Help → View on GitHub opens the project on GitHub: github.com/iannellomarco/hindsight-next.

Open an issue there to report a problem or request a feature. When filing a bug, it helps to include the version you are on — you can find it in the Colophon under The Edition, or in Hindsight → About Hindsight.

Export

Materialize your knowledge as an Obsidian-style vault of linked Markdown notes, or as a knowledge graph in GraphML and Mermaid. Everything stays on your Mac.
